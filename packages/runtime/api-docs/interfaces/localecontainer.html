Interface LocaleContainer

An object storing locale information. It resembles the directory structure that Catalog objects are found in.

Example, if your textdomain is 'myapp':

{
        fr: {
                LC_MESSAGES: {
                           myapp: catalogs['fr']
                }
        },
        'de-DE': {
                LC_MESSAGES: {
                        myapp: catalogs['de-DE']
                }
        }
}
