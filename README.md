# esgettext

GNU gettext-alike runtime and tools for ECMAScript/JavaScript.

## Packages

This mono-repo contains the following packages:

* [runtime](packages/esgettext-runtime/README.md)
* [tools](packages/esgettext-tools/README.md)

## Status

in development.

## Setup

```
$ yarn install
$ yarn run bootstrap
$ yarn run test
```
